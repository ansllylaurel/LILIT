  КАК ЗАПУСТИТЬ БОТА — ПРОСТЫМИ СЛОВАМИ
═══════════════════════════════════════════════════════════════

Что это вообще?
  Это программа-бот для Telegram. Он общается с тобой через нейросеть DeepSeek.
  Чтобы он работал, нужны два "пароля": один от Telegram, другой от DeepSeek.
  Их не покупают — их просто получают бесплатно и вставляют в один файл.

───────────────────────────────────────────────────────────────
ШАГ 0. Установи Python (если ещё нет)
───────────────────────────────────────────────────────────────
  1. Открой в браузере: https://www.python.org/downloads/
  2. Скачай и установи Python (большая жёлтая кнопка).
  3. В самом начале установки ОБЯЗАТЕЛЬНО поставь галочку:
     [✓] Add python.exe to PATH
  4. Нажми "Install Now" и дождись конца. Потом закрой все окна.

───────────────────────────────────────────────────────────────
ШАГ 1. Получить токен бота (пароль от Telegram)
───────────────────────────────────────────────────────────────
  1. Открой Telegram на телефоне или в программе.
  2. В поиске введи: BotFather
  3. Открой чат с BotFather (синяя галочка).
  4. Напиши ему:  /newbot
  5. Он спросит имя бота — напиши любое (например: Мой Помощник).
  6. Потом попросит "username" — придумай короткое имя на английском
     с окончанием bot, например: my_helper_123_bot
  7. В ответ он пришлёт длинную строчку — это и есть ТОКЕН.
     Выглядит так: 7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  8. Скопируй эту строчку целиком (Ctrl+C) и сохрани в блокнот — пригодится.

───────────────────────────────────────────────────────────────
ШАГ 2. Получить ключ DeepSeek (пароль от нейросети)
───────────────────────────────────────────────────────────────
  1. Открой в браузере: https://platform.deepseek.com/
  2. Зарегистрируйся (или войди).
  3. Найди раздел вроде "API Keys" или "Ключи API".
  4. Создай новый ключ (Create API Key). Скопируй его — он показывается один раз.
  5. Сохрани ключ в блокнот.

───────────────────────────────────────────────────────────────
ШАГ 3. Вставить токены в файл .env
───────────────────────────────────────────────────────────────
  1. В папке с ботом найди файл с названием  .env
     (если его нет — сначала один раз запусти install.bat, он создаст .env).
  2. Щёлкни по .env правой кнопкой → Открыть с помощью → Блокнот.
  3. Увидишь что-то вроде:
        TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
        DEEPSEEK_API_KEY=your_deepseek_api_key_here
  4. Замени:
     • your_telegram_bot_token_here — на токен от BotFather (без кавычек и пробелов).
     • your_deepseek_api_key_here — на ключ от DeepSeek.
  5. Сохрани файл: Файл → Сохранить (Ctrl+S). Закрой блокнот.

───────────────────────────────────────────────────────────────
ШАГ 4. Установить программу (один раз)
───────────────────────────────────────────────────────────────
  1. В папке с ботом найди файл  install.bat
  2. Дважды щёлкни по нему.
  3. Откроется чёрное окно, пойдёт установка. Подожди, пока напишет "Готово".
  4. Если напишет "Не найден Python" — значит Python не установлен или не отмечен PATH (вернись к шагу 0).

───────────────────────────────────────────────────────────────
ШАГ 5. Запустить бота
───────────────────────────────────────────────────────────────
  1. В той же папке найди файл  run.bat
  2. Дважды щёлкни по нему.
  3. Откроется чёрное окно с текстом "Бот запущен. Не закрывай это окно."
  4. Пока окно открыто — бот работает. Открой Telegram, найди своего бота по имени и пиши ему.
  5. Чтобы выключить бота — закрой это чёрное окно или нажми в нём Ctrl+C.

───────────────────────────────────────────────────────────────
ЧТО ДЕЛАТЬ, ЕСЛИ НЕ РАБОТАЕТ
───────────────────────────────────────────────────────────────
  • В чёрном окне появилась красная надпись?
    Скопируй её целиком и покажи тому, кто разбирается (или поищи в интернете).
  • Бот не отвечает в Telegram?
    Проверь, что в .env вставлены именно токен от BotFather и ключ от DeepSeek, без лишних пробелов и кавычек.
  • Пишет "TELEGRAM_BOT_TOKEN не задан"?
    Значит файл .env не открыт или не сохранён. Открой .env, вставь токены, сохрани и запусти run.bat снова.

───────────────────────────────────────────────────────────────
КОМАНДЫ БОТА В TELEGRAM
───────────────────────────────────────────────────────────────
  /start  — приветствие и подсказки
  /setrole Ты архитектор  — сделать бота "архитектором" (можно написать любую роль)
  /role   — показать текущую роль
  /reset  — забыть историю и начать разговор заново

  Всё остальное — просто пиши боту текстом, он будет отвечать как чат с нейросетью.
